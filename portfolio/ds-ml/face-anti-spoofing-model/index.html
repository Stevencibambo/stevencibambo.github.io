Project detail

Nowadays, the facial recognition system is used in several applications mainly for in- dividual authentication. Whereas facial recognition remain vulnerable to several types of attacks ; Face Anti-Spoofing detection is a crucial step before providing facial data to the face recognition system. In this work, we are committed to providing a solution to the problem of face anti- spoofing attaque through the using of the Convolutional Neural Network with the dual pixel image sensor. The main goal is to be able to distinguish a reel/genuine face to a fake face as much as possible.
